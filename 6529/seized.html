I extracted the first 288 meme cards of the amazing social, conceptual community of 6529.io. I then used the pixels of each card to create a 12x24 grid. The grid renders a message. Memes seized. This "rememe" is inspired by prior project "RAW MATERIALS" that treats artists as a raw source of data. Pixels extracted, reused to my own purpose, mind pixels blended into new ideas. Mouseover below for the meme card ID. Click to head to 6529.io to view the meme card.

This piece is a 5000x2600 pixel JPG. Zoom to see subtle features of the image showing how I've rearranged some of the pixels from that card. See examples of the grid below the asset. Mouseover for notes.
