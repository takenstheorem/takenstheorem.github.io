The Vessel + VWU

The Vessel is a generative art and on-chain machine project created by Stephen / Imperfect Line and dav where each token (a "craft") carries a dynamic payload that evolves through interactions with on-chain machines. Craft payloads can be crafted, mixed, and transformed via various contracts in the ecosystem. An extensive, expansive sociocultural machine.

Shipyard Work Units are a lightweight, experimental token that rewards craft owners for provable payload evolution. Using symbol VWU, anyone can call the claim function on contract to mint 1 VWU for each craft that has a changed payload.

Visit thevessel.fun to explore the main project and learn more.

How VWU Works

  • Each successful claim of a changed payload mints exactly 1 VWU.
  • Claims are per-craft with a cooldown (currently 1 hour, updatable by the contract owner).
  • If a craft has an associated machine, the claim is keyed off the machine's address; otherwise, it uses the craft's payload hash.
  • First claim per craft always succeeds (previous hash is zero).

Key Details

  • Name: The Vessel Shipyard Work Unit
  • Symbol: VWU
  • Contract Address: 0xfFD29...4A108
  • Decimals: 0 (integer-only by design)
  • Supply: Dynamic; grows with successful claims
  • Claim Cooldown: Configurable (default: 1 hour)

Interacting with VWU

Use the contract's claimWorkUnit(vesselTokenId) or claimWorkUnitBatch(vesselTokenIds[]) functions to mint tokens. Standard ERC-20 functions (transfer, approve, balanceOf, etc.) are supported.

Check a craft's machine with checkMachine(vesselTokenId) to see whether claims will hash the payload or the machine address.

About This Contract

This is an experimental toy token within The Vessel ecosystem. Use at your own risk. The contract is self-contained and does not rely on external oracles for its core logic.
