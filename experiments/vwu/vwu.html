How This Page Behaves

  • Reads balanceOf(address) and totalSupply() from the contract.
  • Calls claimWorkUnitBatch(uint256[]) with the token IDs you enter.
  • Waits for confirmation, then reloads your VWU balance and the token supply.
  • Listens for wallet account and network changes so the page stays in sync.
